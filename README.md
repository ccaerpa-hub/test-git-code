## Work with GIT
1. git checkout main
2. git pull
3. git checkout -b bug-xx-branch-name
4. make changes
5. git add .
6. git commit
7. git checkout main
8. git pull (get latest changes from remote)
9. git checkout bug-xx-branch-name (go back to our local branch)
10. git merge main (merge you local changes in main)
11. git checkout main
12. git merge bug-xx-branch-name (now finally merge evnerything back into local main branch)
13. git pull (pull main from remote once more to make sure notjhing else has changed)
14. git push (now push local main branch to remote)
15. git branch -d bug-xx-branch-name (dlete local branch as not needed anymore)
